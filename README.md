# 📊 EDA on Retail Sales Data

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on retail sales data to uncover patterns, trends, and insights that help in business decision-making.

## 🎯 Objectives
- Analyze sales data
- Identify trends and patterns
- Understand customer behavior
- Provide actionable insights

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Key Insights
- Electronics is the top-performing category
- Sales vary across months (seasonal trends)
- Female customers slightly contribute more
- Middle-aged customers spend more
- Top customers generate high revenue

## 📁 Files
- EDA_Retail_Sales_Analysis.ipynb
- Retail Sales EDA Report (PDF)

## 🚀 Conclusion
The project provides valuable insights into customer behavior and sales performance, helping businesses make data-driven decisions.
